FEATURES

核心功能

從日常記錄到數據分析，涵蓋個人閱讀管理的完整工作流程。

📝

書籍筆記 CRUD

新增、瀏覽、編輯、刪除書籍記錄。支援封面圖片、評分、心得、引用金句等完整欄位。

🔍

即時搜尋與篩選

全文搜尋書名、作者、心得內容。支援類別、評分範圍、日期區間多條件組合篩選。

📊

統計儀表板

視覺化閱讀統計，包含每月趨勢折線圖、類別分佈圓餅圖、Top 3 高分書籍。

💬

金句收藏庫

集中展示所有書籍引用金句，支援複製至剪貼簿與「今日隨機金句」功能。

🏷️

標籤管理

自訂書籍分類標籤，支援新增、重新命名、刪除。系統提供 6 個預設分類標籤。

⬇️

資料匯出備份

一鍵匯出所有書籍記錄為 CSV（可用 Excel 開啟）或 JSON 格式，檔名自動帶入日期。
