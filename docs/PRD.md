## 1. 產品概覽

### 1.1 產品願景

打造一個個人專屬的數位讀書筆記本，讓使用者能夠輕鬆記錄閱讀心得、管理書單，並透過搜尋與統計功能回顧閱讀軌跡，養成良好的閱讀習慣。

### 1.2 問題陳述

讀完一本書後，心得與感悟往往隨時間流逝而淡忘。紙本筆記不易搜尋，散落在各處的記錄也難以整理。本系統旨在解決以下痛點：

- 閱讀心得無系統化記錄，容易遺忘
- 無法快速查找過去讀過哪些書、評分如何
- 無法掌握自己的閱讀習慣與偏好

### 1.3 產品範疇

本系統為**個人單機/本地端 Web 應用程式**，無需多用戶功能，以簡潔、高效的使用體驗為優先。

## 3. 核心功能需求

### 功能 1：新增書籍筆記

**描述**：使用者可以新增一筆讀書記錄，填寫書籍資訊與個人心得。

**輸入欄位**：

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 書名 | 文字 | ✅ | 書籍完整名稱 |
| 作者 | 文字 | ✅ | 作者姓名（可多位） |
| 閱讀完成日期 | 日期 | ✅ | 實際讀完的日期 |
| 個人評分 | 數值 | ✅ | 1～5 星（支援半星） |
| 閱讀心得 | 長文字 | ✅ | 個人感想、重點摘要 |
| 書籍類別 | 標籤/分類 | ❌ | 例：科技、文學、心理、商業 |
| 封面圖片 URL | 文字 | ❌ | 書籍封面連結（或本地上傳） |
| 是否公開推薦 | 布林值 | ❌ | 標記是否想推薦給他人 |
| 引用金句 | 長文字 | ❌ | 摘錄書中印象深刻的句子 |

**驗收標準**：
- 必填欄位未完成時，顯示錯誤提示，禁止送出
- 成功新增後，自動跳轉至該筆記詳細頁面
- 新增成功顯示 Toast 通知

---

### 功能 2：檢視與編輯書籍筆記

**描述**：使用者可以瀏覽所有書籍筆記清單，並進入單筆筆記進行編輯或刪除。

**清單頁功能**：
- 以卡片形式呈現書名、作者、評分、閱讀日期
- 支援依「閱讀日期」、「評分高低」、「書名字母順序」排序
- 顯示書籍類別標籤

**詳細頁功能**：
- 完整顯示所有欄位資訊
- 提供「編輯」按鈕，進入編輯模式（表單預填原始資料）
- 提供「刪除」按鈕，附確認對話框防止誤刪

**驗收標準**：
- 編輯儲存後，詳細頁即時更新
- 刪除後返回清單頁，清單即時更新

---

### 功能 3：搜尋與篩選

**描述**：使用者能以多種條件快速找到目標書籍。

**搜尋功能**：
- **全文搜尋**：同時搜尋書名、作者、心得內容中的關鍵字
- **即時搜尋**：輸入時即時顯示結果（不需按 Enter）
- **關鍵字高亮**：搜尋結果中標記符合的文字

**篩選功能**：

| 篩選條件 | 選項 |
|---------|------|
| 書籍類別 | 多選標籤 |
| 評分範圍 | 滑桿（1～5 星） |
| 閱讀日期區間 | 起始日～結束日 |
| 是否公開推薦 | 是 / 否 / 全部 |

**驗收標準**：
- 搜尋與篩選可同時組合使用
- 無結果時顯示友善的空狀態提示
- 清除搜尋條件後，恢復顯示全部書單

---

### 功能 4：閱讀統計儀表板

**描述**：提供視覺化統計，讓使用者一覽自己的閱讀概況。

**統計指標**：

| 指標 | 說明 |
|------|------|
| 總閱讀書籍數 | 所有記錄的書籍總數 |
| 本月閱讀數 | 當月完成的書籍數量 |
| 本年閱讀數 | 當年完成的書籍數量 |
| 平均評分 | 所有書籍評分的平均值 |
| 最高評分書籍 | Top 3 高分書籍列表 |
| 閱讀類別分佈 | 各類別書籍數量（圓餅圖） |
| 每月閱讀趨勢 | 過去 12 個月閱讀量（折線圖） |
| 閱讀頁數統計 | 若有記錄頁數，顯示總頁數（選配） |

**驗收標準**：
- 圖表使用 Chart.js 或類似套件呈現
- 統計數據即時反映最新資料
- 儀表板提供「今年」、「全部時間」等時間切換

---

### 功能 5：標籤與分類管理

**描述**：使用者可以自訂書籍分類標籤，並以標籤瀏覽書單。

**功能細節**：
- 系統預設標籤（科技、文學、心理學、商業、歷史、其他）
- 使用者可新增自訂標籤
- 可對標籤進行重新命名與刪除（刪除時確認是否影響現有書籍）
- 標籤頁面：點擊標籤，篩選該標籤下的所有書籍

**驗收標準**：
- 標籤名稱不可重複
- 刪除標籤後，原本含有該標籤的書籍記錄不受損（標籤欄清空）

---

### 功能 6：引用金句收藏庫

**描述**：集中顯示所有書籍中摘錄的引用金句，讓使用者隨時回味。

**功能細節**：
- 獨立的「金句收藏」頁面
- 以卡片形式展示金句內容、出處書名、作者
- 支援隨機顯示一則金句（「今日金句」功能）
- 可複製金句文字至剪貼簿

**驗收標準**：
- 金句收藏頁僅顯示有填寫金句的書籍記錄
- 複製成功後顯示提示訊息

---

### 功能 7：資料匯出與備份

**描述**：使用者可將所有讀書記錄匯出，保存備份或於其他裝置使用。

**匯出格式**：
- **CSV**：可用 Excel 開啟的試算表格式
- **JSON**：結構化資料，可用於備份與未來匯入

**驗收標準**：
- 匯出的 CSV 包含所有欄位（評分、心得、日期等）
- 匯出時間戳記自動帶入檔名（例：`reading_notes_20260425.csv`）

## 6. 資料模型（草案）

### Books（書籍筆記）

```sql
CREATE TABLE books (
    id           INTEGER PRIMARY KEY AUTOINCREMENT,
    title        TEXT    NOT NULL,
    author       TEXT    NOT NULL,
    finished_at  DATE    NOT NULL,
    rating       REAL    NOT NULL CHECK(rating BETWEEN 1 AND 5),
    review       TEXT    NOT NULL,
    cover_url    TEXT,
    is_recommend BOOLEAN DEFAULT FALSE,
    quote        TEXT,
    created_at   DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at   DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### Tags（標籤）

```sql
CREATE TABLE tags (
    id   INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE
);
```

### Book_Tags（多對多關聯）

```sql
CREATE TABLE book_tags (
    book_id INTEGER REFERENCES books(id) ON DELETE CASCADE,
    tag_id  INTEGER REFERENCES tags(id)  ON DELETE SET NULL,
    PRIMARY KEY (book_id, tag_id)
);
```

## 7. UI/UX 需求

- **設計風格**：深色模式（Dark Mode）為主，現代簡潔
- **字型**：使用 Google Fonts（建議 `Inter` 或 `Noto Sans TC`）
- **配色**：以深藍/深灰為主色，金色/琥珀色為強調色（書籍主題感）
- **動畫**：頁面切換淡入、卡片 hover 浮起效果
- **導航**：左側側邊欄導航，包含：書單、新增筆記、搜尋、統計、金句、標籤管理

## 9. 開放問題（待確認）

> [!IMPORTANT]
> 以下問題請在開始開發前確認：

1. **封面圖片**：是否需要支援**本地上傳**圖片，還是僅填寫外部 URL 即可？
2. **評分精度**：評分是否支援半星（例如 3.5 星），還是僅整數 1~5？
3. **頁數記錄**：是否需要記錄書籍總頁數（用於統計總閱讀頁數）？
4. **匯入功能**：是否需要**匯入** CSV/JSON 功能（將舊資料帶入系統）？
5. **手機支援**：是否需要在手機上操作？（若需要，響應式設計優先級提升）
