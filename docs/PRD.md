## 1. 專案概述

### 1.1 背景與動機

閱讀後若沒有系統化地記錄心得，知識很快就會遺忘。現有的手寫筆記或零散的文字檔缺乏搜尋、分類與評分功能，難以回顧與比較。本專案旨在建立一個個人專屬的讀書筆記本系統，讓閱讀心得能夠被有效管理與快速查找。

### 1.2 目標用戶

- **主要用戶**：本人（個人使用）

### 1.3 核心價值主張

> 一個輕量、快速、易用的個人讀書筆記管理工具，讓每一本讀過的書都能留下有價值的記錄，並隨時可被搜尋與回顧。

### 2.2 使用者故事

#### F1 — 新增書籍筆記
> 作為**讀者**，我希望**能新增一筆書籍記錄（包含書名、作者、閱讀心得、評分）**，以便**將剛讀完的書留下完整記錄**。

**驗收條件：**
- 表單欄位：書名（必填）、作者（選填）、心得（必填）、評分（1–5）、閱讀日期、標籤
- 成功送出後跳轉至詳情頁並顯示成功提示

---

#### F2 — 瀏覽所有筆記（列表頁）
> 作為**讀者**，我希望**能看到所有筆記的列表，每筆顯示書名、評分與摘要**，以便**快速掌握我讀過哪些書**。

**驗收條件：**
- 顯示書名、作者、評分（星星）、閱讀日期
- 支援按日期、評分排序
- 每頁顯示最多 10 筆，支援分頁

---

#### F3 — 搜尋筆記
> 作為**讀者**，我希望**能輸入關鍵字搜尋書名或心得內容**，以便**快速找到某本書的筆記**。

**驗收條件：**
- 搜尋範圍：書名、作者、心得內容
- 搜尋結果即時顯示並標示匹配關鍵字
- 無結果時顯示「查無筆記」提示

---

#### F4 — 編輯筆記
> 作為**讀者**，我希望**能修改已建立的筆記內容**，以便**補充或修正之前記錄的心得**。

**驗收條件：**
- 編輯頁預填原始資料
- 儲存後返回詳情頁並顯示「更新成功」

---

#### F5 — 刪除筆記
> 作為**讀者**，我希望**能刪除不需要的筆記**，以便**保持筆記本的整潔**。

**驗收條件：**
- 刪除前彈出確認對話框
- 刪除後跳轉至列表頁並顯示「已刪除」提示

---

#### F6 — 書籍評分（1–5 星）
> 作為**讀者**，我希望**能為每本書打上 1 到 5 顆星的評分**，以便**快速比較不同書籍的推薦程度**。

**驗收條件：**
- 以互動式星星 UI 顯示評分
- 列表頁顯示評分，可按照評分篩選

---

#### F7 — 標籤／分類管理
> 作為**讀者**，我希望**能為筆記新增自訂標籤（如：心理學、小說、工具書）**，以便**依主題分類瀏覽筆記**。

**驗收條件：**
- 支援多標籤輸入
- 列表頁可依標籤篩選

---

#### F8 — 閱讀狀態追蹤
> 作為**讀者**，我希望**能標記書籍的閱讀狀態（想讀 / 閱讀中 / 已讀完）**，以便**追蹤閱讀進度**。

**驗收條件：**
- 狀態以顏色 badge 顯示（想讀=灰、閱讀中=藍、已讀完=綠）
- 可在列表頁依狀態篩選

---

#### F9 — 統計儀表板
> 作為**讀者**，我希望**能看到閱讀統計（今年讀了幾本、平均評分、最常見標籤）**，以便**了解自己的閱讀習慣**。

**驗收條件：**
- 顯示：總筆記數、今年閱讀數、平均評分、標籤雲

---

#### F10 — 筆記匯出
> 作為**讀者**，我希望**能將所有筆記匯出為 CSV 或 Markdown 格式**，以便**備份或在其他工具使用**。

**驗收條件：**
- 提供「匯出 CSV」與「匯出 Markdown」按鈕
- 匯出檔案包含所有欄位

## 3. 非功能需求

### 3.1 技術限制

| 項目 | 選用技術 |
|------|----------|
| 後端框架 | Python Flask |
| 模板引擎 | Jinja2 |
| 資料庫 | SQLite |
| 前端樣式 | Vanilla CSS（可搭配少量 JS） |
| 部署環境 | 本機執行（localhost） |

### 3.2 效能考量

- 搜尋回應時間 < 500ms（資料量 < 1000 筆）
- 頁面首次載入 < 2 秒

### 3.3 安全考量

- 資料庫使用 ORM 或參數化查詢，防止 SQL Injection
- 表單輸入進行基本驗證與 XSS 防護
- 本機使用，無需身份驗證模組

## 附錄：資料欄位草稿

### 書籍筆記（Book Note）

| 欄位名稱 | 型態 | 說明 |
|----------|------|------|
| id | INTEGER | 主鍵，自動遞增 |
| title | TEXT | 書名（必填） |
| author | TEXT | 作者（選填） |
| content | TEXT | 閱讀心得（必填） |
| rating | INTEGER | 評分 1–5（選填） |
| status | TEXT | 想讀 / 閱讀中 / 已讀完 |
| read_date | DATE | 閱讀完成日期 |
| tags | TEXT | 標籤（逗號分隔） |
| created_at | DATETIME | 建立時間 |
| updated_at | DATETIME | 最後修改時間 |
